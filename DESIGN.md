# Design: Jitin Nair Portfolio

## Color Strategy: Committed (Emerald)
- Background: `oklch(12.72% 0.0053 285.83)` (Deep Dark Indigo/Black)
- Accent (Primary): `oklch(76.84% 0.16 160.0)` (Emerald 400)
- Surface: `oklch(18.0% 0.0053 285.83)`
- Border: `oklch(25.0% 0.0053 285.83 / 0.1)`

## Typography
- Primary: Geist / Satoshi (Sans-serif)
- Secondary: Serif (for highlights/quotes)
- Base Size: 16px

## Principles
- Asymmetric Layouts (Bento 2.0)
- 1px Borders
- Subtle Grain
- Liquid Glass Refraction
- Perpetual Micro-interactions
