# Prompt Engineering Playbook
## By Jitin Nair — AI Systems Architect

### 10 Battle-Tested Prompt Patterns

**1. Chain-of-Thought (CoT)**
Ask the model to think step-by-step before answering.

**2. Few-Shot Examples**
Provide 2-3 examples of desired input/output format.

**3. Role-Based Prompting**
Assign a specific expert role before the task.

**4. Structured Output**
Request JSON, XML, or markdown formatted responses.

**5. Self-Consistency**
Generate multiple responses and pick the majority answer.

**6. ReAct (Reasoning + Acting)**
Combine reasoning traces with action steps.

**7. Constitutional AI**
Add self-critique steps to reduce harmful outputs.

**8. Chain-of-Verification**
Ask the model to verify its own claims before finalizing.

**9. Decomposition**
Break complex tasks into sub-tasks with explicit handoffs.

**10. Meta-Prompting**
Ask the model to write a better prompt for the task.
